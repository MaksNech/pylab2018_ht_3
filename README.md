# Minesweeper game.

## 1: Initial Setup

### Clone project in a new directory:
```bash
cd path/to/a/new/directory
git clone https://github.com/MaksNech/pylab2018_ht_3.git
```

## 2: Getting Started

### Start backend:
Inside project directory run app with terminal command:
```bash
python3 main.py
```
